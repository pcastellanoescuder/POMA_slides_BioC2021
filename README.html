POMA slides for the BioC2021 virtual conference

This repository contains the POMA slides for the BioC2021 conference.

BioC2021 is a virtual conference (from Aug 4, 2021 to Aug 6, 2021) aimed at biologists, bioinformaticians, statisticians, programmers and software engineers who use or contribute to the Bioconductor project, or are interested in using Bioconductor.
